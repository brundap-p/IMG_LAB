# Image Processing Laboratory Programs using OpenCV

This repository contains Image Processing laboratory programs completed as part of my postgraduate coursework. The programs demonstrate fundamental image processing techniques and computer vision operations using OpenCV and Python.

The laboratory exercises cover various image processing concepts, including image enhancement, filtering, transformations, edge detection, thresholding, segmentation, and feature extraction.

## Technologies Used

* Python
* OpenCV
* NumPy
* Matplotlib
* Jupyter Notebook

## Topics Covered

* Image Reading and Display
* Image Transformations
* Image Filtering and Smoothing
* Histogram Processing
* Thresholding Techniques
* Edge Detection
* Morphological Operations
* Image Segmentation
* Feature Extraction

## Purpose

The objective of this repository is to document and showcase the Image Processing laboratory exercises completed during my postgraduate academic curriculum.
